# 🏏 IPL Dashboard 2023–2026 | Power BI

An interactive Power BI dashboard that provides a comprehensive analysis of the Indian Premier League (IPL) seasons from 2023 to 2026. The dashboard transforms raw match data into meaningful insights through dynamic visualizations, KPIs, and interactive filters, helping users analyze team performance, season statistics, and tournament trends.

## 🚀 Features

- 📊 Executive Overview Dashboard
- 🏏 Team Performance Analysis
- 📈 Match & Season Statistics
- 🏆 Winning Percentage Analysis
- 📅 Season-wise Performance Comparison
- 🎯 Interactive Filters (Season & Team)
- 📌 Dynamic KPIs
- 🎨 Clean and User-Friendly Dashboard Design

## 📊 Key KPIs

- Total Matches
- Total Runs
- Total Wickets
- Highest Score
- Total Seasons
- Total Teams

## 📈 Dashboard Insights

- Team-wise Total Match Wins
- Winning Percentage Comparison
- Total Matches by Year
- Total Runs by Year
- Team Performance Summary
- Season Champions (2023–2026)

## 🛠️ Tools & Technologies

- Power BI Desktop
- Power Query
- DAX
- Data Modeling

## 🎯 Objective

The objective of this project is to analyze IPL data from 2023–2026, compare team performances across seasons, identify winning trends, and present actionable insights through an interactive Power BI dashboard.
